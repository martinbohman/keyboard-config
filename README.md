# keyboard-config

## Banana Split 60

Usage:

```
sudo dfu-programmer atmega32u4 erase --force
sudo dfu-programmer atmega32u4 flash bananasplit/tmk_keymap.hex
sudo dfu-programmer atmega32u4 reset
```
